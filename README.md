# Personal Finance Project
***
This program is made to be a way to keep track of incomes and expenses, giving dates and categories for them. It includes the ability to search for incomes and or expenses by date, aswell as searching for expenses by category. It also includes functionality to view total income or expenses within a certain time period. It contains a pie chart that shows the cost of expenses and their percents.

## How to use
***
1. Pygame is needed, and matplotlib
2. To install either, you will want to open the terminal, and type "pip install pygame" for pygame. for matplotlib, replace pygame with matplotlib in the thing.
3. After you have both installed, you should be able to run the program from main.

## Details on Project features
***
- This features a pygame terminal, and matplotlib to make a piechart
- It allows you to enter incomes and expenses, giving them dates, and categories for the expenses.
- It allows you to search for incomes and or expenses within a given time period, and see the total of either, or the net positive or negative.
